SmartHome App

Android-приложение для управления умным домом через Bluetooth с поддержкой IoT-устройств на базе Atmega328p

SmartHome App — это современное решение для управления климатом, освещением и датчиками в системе умного дома. Проект включает Android-приложение, разработанную Bluetooth-библиотеку и прошивку для микроконтроллера с двунаправленным JSON-протоколом

Основные функции:

• Мониторинг температуры, влажности и давления через Bluetooth SPP
• Настройка RGB-подсветки с режимами: моноцвет, композиция из палитры изображений (с использованием Palette API), автоматические сценарии
• Акустическая обратная связь и LED-индикация на устройстве
• Анализ изображений для генерации цветовых схем освещения
• Реактивное обновление данных через SharedFlow и Coroutines
• Автоматическое отключение ресивера при простое
• Поддержка светлой/тёмной тем, трёх языков (русский, английский, китайский)
• Вертикальный и горизонтальный режимы экрана
• Анимации и адаптивная вёрстка (FlexboxLayout, ViewPager2)

Технологический стек:

• Языки: Kotlin (Android), C/C++ (прошивка Atmega328p)
• Архитектура: Clean Architecture + MVVM, многомодульная структура
• Koin (Dependency Injection), Room, SharedPreferences
• Собственная Bluetooth-библиотека с поддержкой потоковой передачи данных
• Bluetooth SPP, сокеты
• Протокол взаимодействия на основе JSON с оптимизацией для микроконтроллеров (примеры сообщений: LED_MONO, TIME_UPDATE, COM_ENABLED)
• Прошивка устройства с обработкой сенсоров и управлением RGB-лентой
• Glide, Palette API, Jetpack Navigation Component
• Gradle + Proguard для оптимизации сборки
