SwApi App

Интерактивный гид по вселенной «Звёздных войн» на вашем Android-устройстве

SwApi App — это современное Android-приложение для поиска и просмотра информации о культовой кино-вселенной «Звёздные войны» с использованием открытого REST API SWAPI (https://swapi.dev/)

Приложение реализовано с применением актуальных архитектурных подходов и демонстрирует современные практики разработки мобильных приложений на Kotlin

Основные функции:

• Поиск персонажей, фильмов, планет, звездолётов и других объектов вселенной Star Wars
• Просмотр подробной информации о каждом объекте с удобной навигацией по категориям
• Быстрая загрузка и отображение изображений
• Кэширование данных для офлайн-доступа
• Интуитивно понятный интерфейс с поддержкой BottomNavigationView и вложенной навигацией
• Адаптивная архитектура, позволяющая легко масштабировать и расширять функциональность приложения

Технологический стек:

• Язык: Kotlin
• Архитектура: Clean Architecture, MVVM и многомодульность для разделения ответственности и удобной поддержки
• REST API: интеграция с SWAPI через Retrofit для надёжного обмена данными
• Хранение данных: Room для локального кэширования и быстрого доступа к информации
• Внедрение зависимостей: Koin - лёгкий DI-фреймворк для управления зависимостями
• Навигация: Single Activity и Navigation Architecture Component (NavController) для управления переходами и вложенными графами навигации
• Glide, LiveData, Gson

Назначение приложения:

SwApiApp разработан в образовательных целях для демонстрации современных подходов к созданию Android-приложений и освоения работы с открытыми API
