GitHubClient App

Клиентское приложение для работы с сервисом GitHub

GitHubClient App — это учебный пример Android-приложения для взаимодействия с GitHub REST API (https://docs.github.com/en/rest), разработанное с применением актуальных технологий и архитектурных подходов. Проект демонстрирует современные практики создания мобильных клиентов для работы с облачными сервисами

Основные функции:

• Просмотр публичной информации о репозиториях, пользователях и активности через REST API
• OAuth-авторизация с использованием токена
• Настройки светлой/тёмной тем и выбора языка (русский, английский и китайский)

Технологический стек:

• Язык: Kotlin
• OAuth аутентификация и авторизация с помощью токена
• Gson, Glide и Retrofit
