Search Job

Приложение поиска и добавления в избранное вакансий сервиса HeadHunter

Search Job — это Android приложение для поиска и организации вакансий с использованием API HeadHunter (https://dev.hh.ru)

Основные функции:

• Поиск вакансий: поиск актуальных вакансий в реальном времени используя API HeadHunter
• Фильтр поиска: настройка фильтрации поиска по профессии, месту работы и уровню зарплаты
• Избранные вакансии: добавление вакансий в избранное для быстрого доступа
• Экран описания вакансии: информация о вакансии
• Темы оформления: автоматическое переключение между светлой и тёмной темами

Технологический стек:

• Язык Kotlin
• Modularization, Clean Architecture, MVVM и Single Activity
• REST API HeadHunter
• Coroutines
• SQLite, Room и SharedPreferences для хранения данных
• Koin, Glide, Gson, LiveData, Retrofit и OkHttp

Назначение приложения:

Разработано в составе команды разработчиков в рамках образовательного проекта в Яндекс.Практикум (https://practicum.yandex.ru/)
