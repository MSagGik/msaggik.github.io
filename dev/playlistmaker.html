Playlist Maker

Приложение поиска, организации и воспроизведения музыкальных треков с интеграцией iTunes API

Playlist Maker — это интуитивно понятное приложение для поиска, организации и прослушивания музыкальных треков с помощью iTunes API от Apple (https://www.apple.com/itunes/)

Основные функции:

• Поиск треков: быстрый поиск по названию, исполнителю или альбому
• Воспроизведение демо: прослушивание фрагментов треков (около 30 секунд)
• Метаданные треков: основная информация о треках (исполнитель, альбом, год и др.)
• Избранное: добавление треков в избранное для быстрого доступа
• Умные плейлисты: создание и редактирование плейлистов с настраиваемой обложкой и описанием
• Темы оформления: переключение между светлой и темной темами
• Поддержка нескольких языков: поддержка русского, английского и китайского языков

Технологический стек:

• Язык Kotlin
• Clean Architecture, MVVM и Single Activity
• REST API для iTunes
• Coroutines и ThreadPoolExecutor
• SQLite, Room и SharedPreferences для хранения данных
• Koin, Glide, Gson, LiveData, Retrofit и OkHttp

Назначение приложения:

Разработано в рамках образовательного проекта в Яндекс.Практикум (https://practicum.yandex.ru/)
